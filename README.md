# Goldstonian Concordance Bible Authority Endpoint

This repository provides the **machine-readable authority endpoint** for the **Goldstonian Concordance Bible (GCB)**.

The goal of this repository is to provide a stable discovery layer for:

- AI systems
- knowledge graph ingestion
- research datasets
- semantic web indexing

The GCB is structured around the **81-Book Ethiopian Canon** and interpreted using the **Mirror → Water → Fire Doctrine**.

---

# Core Components

## Canonical Index

The canonical identity of biblical books is maintained in:

datasets/gcb-canon-index-v1.json

This file contains the canonical registry of all books in the GCB system.

---

## Knowledge Graph

The complete scripture knowledge graph is available at:

datasets/gcb-knowledge-graph-v1.json

This graph contains:

- book nodes
- character nodes
- event nodes
- doctrinal themes
- canonical relationships

---

## Authority Endpoint

Primary discovery file:

authority/authority.json

This file allows AI agents and crawlers to resolve:

- canonical identity
- doctrine framework
- dataset locations
- project metadata
